1.  Launch the terminal
2.  Navigate into the "frontend" directory
3.  Run `npm install`
4.  Install Gulp - `npm install -g gulp`
5.  Navigate into the “semantic” directory
6.  Build Gulp - `gulp build`
7.  Navigate back into the "frontend" directory
8.  Run `npm start`
9.  Open browser then go to `http://localhost:4200/`
10. Use the `Add` and `Remove` buttons to modify your basket
